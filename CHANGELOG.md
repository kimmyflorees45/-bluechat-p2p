# Changelog - BlueChat P2P

## [1.0.0] - 2025-11-07

### Added
- Implementación inicial de BlueChat P2P
- Descubrimiento automático de dispositivos Bluetooth
- Filtrado inteligente para mostrar solo celulares
- Chat en tiempo real con stickers
- Sistema de conexión robusto con reintentos
- Interfaz personalizable con colores
- Menú hamburguesa funcional
- Guardado de preferencias del usuario

### Fixed
- Problemas de compatibilidad con Android 12+
- Errores de conexión Bluetooth
- Filtrado de dispositivos no móviles
- Manejo de permisos en diferentes versiones de Android

### Technical
- Flutter 3.16+
- flutter_bluetooth_serial 0.4.0
- shared_preferences 2.2.2
- Material Design 3
